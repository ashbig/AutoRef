Running ACE

Procedures to get ACE running:

  • Put ACE.war file at the directory of TOMCAT_HOME/webapps
  • Start-up TOMCAT
  • Stop TOMCAT
  • Go to TOMCAT_HOME/webapps to find the folder ACE generated by running TOMCAT
  • Find the web.xml file at the directory of TOMCAT_HOME/webapps/ACE/WEB-INF
  • Open web.xml file with text editor programm to modify it to reflect your database location and credentials.

    <!-- for oracle database -->
    <init-param>
          <param-name>jdbcURL</param-name>
          <param-value>your database URL</param-value>
    </init-param>
    <init-param>
          <param-name>user</param-name>
          <param-value>your_user_name</param-value>
    </init-param>
    <init-param>
          <param-name>password</param-name>
           <param-value>your_password</param-value>
    </init-param>
    <!-- for oracle database -->

  • Modify ApplicationHostSettings.properties file from TOMCAT_HOME/webapps/ACE/WEB-INF/classes/config directory with text editor programm.
  • Modify ModuleSettings.properties file from TOMCAT_HOME/webapps/ACE/WEB-INF/classes/config directory with text editor.
  • Start-up TOMCAT server again.
  • Try to register to verify that application is working
