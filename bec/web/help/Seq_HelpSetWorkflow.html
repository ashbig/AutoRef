Process
Description
Run sequencing for end reads

User: selects what reads to run (forward/reverse/both )

System:

  • Creates new plates for end reads (one plate copy per read type)
  • Creates naming file and sends it to user by e-mail
Run end read wrapper Fore execution of end reads wrapper (file transfer, Phred base call, reads information parsing)
Check whether all reads are available System checks that all required runs to be complete (for plates that requires both reads to be run only)
1. Run isolate ranker
2. Run polymorphism finder for isolate ranking

User:

  • select settings for isolate ranker
  • select plates(reads originated from the same glycerol plate selected as one entity)

System:

  • executes isolate ranker
  • executes polymorphism finder (optional, but can not be run separately)
View, change (optional) and approve isolate ranking User: selects plate name

System: present output of isolate ranker to the user
Send sequencing plates to the outside facility

System:

  • Create sequencing plates (can rearray plates)
  • creates files with sequence information to be send with plate to the sequencing facility
Receive sequencing plates from the outside facility

User: notify user what plates came back

System: changes plate location
Upload data from outside facility System: upload assembled sequences from file. If uploaded data are for plates that has been not analyzed by ACE for end reads, sequencing plates are created.
Run primer designer

User: select plates for full sequencing (system should know what sequences from these plates were covered by end reads)

System:

  • Design primers
  • Create template plates
  • Create oligo plates
  • Create file for oligo order
Receive oligo plates

User: notify system what plates came back

System: changes plate location
Run full sequencing process User: select template plates

System: creates naming files
Run assembly wrapper Forse execution of assembly wrapper (file transfer, assembly, sequence information parsing)
Run discrepancy finder for clone evaluation

User:

  • select settings
  • select template plates

System: run discrepancy finder for isolates on selected plates

Run polymorphism finder for clone evaluation

User:

  • select settings
  • select template plates

System: run polymorphism finder for isolates on selected plates

Run decision tool

User:

  • select settings
  • select template plates

System: run decision tool for isolates on selected plates and provide results to user

User: approves results with changes (optional)

System: send rearray file to user
Storage plate assembly
