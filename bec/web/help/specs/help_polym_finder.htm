Polymorphism Finder Specification


Polymorphism Finder is a two step process. First, a relatively short sequence segment comprising the discrepancy and its flanking nucleotides is compared using NCBI BLAST to data from all user-selected GenBank databases to find an identical match. The number of nucleotides in flanking sequence defined by 'Number of flanking bases to append on both sides of discrepancy for search' parameter, however, it should be less or equal to 20. The list of available GenBank databases on local non-ACE server is defined by administrator in ACE configuration file as set of paired parameters (PF_DB_1...N - PF_DB_1...N_NAME). Second step is verification of each 100% hit by comparing the entire clone target sequence with the hit sequence using Pairwise BLAST. The resulting alignment should satisfy 'minimum match length' and 'required percent identity' requirements to confirm polymorphism.
