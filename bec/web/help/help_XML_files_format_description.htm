XML format for plate upload

Reference sequence file

Reference sequence file describs target sequences for clones. sequence element contains: sequence id, sequence, species, CDS start, CDS stop, cDNA source, chromosome and sequence properties (terms in bold are required). The sequence id is the unique identifier for each sequence and will be used as 'user reference sequence id'. Each sequence can be described by features. The feature element description contains: feature name type ('GenBank Accession'), feature name value (the GenBank accession number for 'GenBank Accession' name type), feature url and description. The annotation type field must use valid annotation types from the database. Please define all annotation types prior to reference sequence submission.
