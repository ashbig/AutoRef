Help :General Report

Tool goal:
Report Runner is a tool designed to assist the researcher in accessing information about available clones. The output tab delimited file is send by e-mail to user. There is no limit on number of submitted items.
Information provided by report
Field Name
Description
  Infomation type: Clone Information:
Clone ID Clone ID is a unique clone identifier shared by ACE and FLEX
Clone final status Possible values: 'In Process', 'Accepted', 'Rejected'. If clone final status set by user to 'Accepted' or 'Rejected', clone data cannot be modified.
Sample ID ACE sample Id
Plate label Plate label
Sample type Sample Type (Isolate, Empty, Control)
Sample well Position on the plate ( 1 - 96 )
Clone score Clone penalty score, calculated
Clone rank Clone Rank, defines whether clone (a) match expected ORF; (b) acceptable based on user criteria; (c) clone quality compare to the other clones originated from the same isolate.
 
  Infomation type: Clone raw data information
Directory name Name of clone trace files directory.
End reads length( high quality region) Length of the high confidence region for forward and reverse end reads
Status of clone sequence assembly Status shows: (a) clone went throw assembly process; (b) was assembly successes or not; (c) reason for assembly failure.
 
Infomation type: Reference Sequence Information
Sequence ID ACE/Flex sequence Id, both values specified
CDS start CDS start
CDS stop CDS stop
CDS length CDS length
GC Content GC content of gene sequence
Sequence Text Gene sequence (text) as was defined in FLEX on reference sequence upload
CDS Gene CDS (text) as was defined in FLEX on reference sequence upload
GI Number GI as was stored in FLEX on reference sequence upload
Gene Symbol Gene symbol, as was stored in FLEX on reference sequence upload
Species specific ID Species specific identifier as was defined by user.
All available identifiers All identifiers for reference sequence, except for species specific identifier.
Infomation type: Clone Sequence Information
Clone sequence ID Clone sequence ID, if clone does not have fully assembled sequence ID equals to 0.
