General Report

Tool goal:
General Report is a tool designed to assist the researcher in accessing information about available clones. The output tab delimited file is send by e-mail to user. There is no limit on number of submitted items.
Information provided by report
Field Name
Description
  Information type: Clone Information:
Clone ID Clone ID is a unique clone identifier shared by ACE
Clone final status Possible values: 'In Process', 'Accepted', 'Rejected'. If clone final status set by user to 'Accepted' or 'Rejected', clone data cannot be modified
Sample ID ACE sample ID
Plate label Plate label
Sample type Sample Type (Isolate, Empty, Control)
Sample well Position on plate ( 1 - 96 )
Clone score Clone penalty score, calculated
Clone rank Clone Rank, defines whether clone (a) match expected ORF; (b) acceptable based on user criteria; (c) clone quality compares with other clones originating from the same isolate
 
  Information type: Clone raw data information
Directory name Name of clone trace file directory
End read length ( high quality region) Length of high confidence region for forward and reverse end read
Status of clone sequence assembly Status shows: (a) whether clone went throw assembly process; (b) assembly succeeded or not; (c) reason for assembly failure
